en temps réel

Les mesures ci-dessus sont synchronisées avec votre position grâce à la géolocalisation du navigateur, et interrogent l'API météo en direct. Vous pouvez aussi basculer sur les mesures envoyées par la carte NUCLEO-L476RG branchée en local.

Localisation en cours…

Connexion à la carte (Web Serial)

Branchez la NUCLEO-L476RG en USB, flashez-la avec un firmware qui envoie une ligne JSON par mesure sur le port série (ex. via VitaScience), puis cliquez sur connecter. Format attendu : {"temp":21.4,"hum":55,"press":1014,"wind":12,"prec":0}

Matériel

Le matériel & Logiciel
utilisé pour cette station

Pour réussir ce projet de station météo, j'ai utilé du codage et certains outils pour simulé des compteurs météo.

Visual Studio Code

Application de programation, qui permet d'exécuter n'importe quelle language de code.

NUCLEO-L476RG

Outil permettant de simuler un thermomètre qui affichait la température et l'humidité.

VitaScience

Logiciel de programation permettant d'entrer un code dans le NUCLEO afin d'avoir de l'affichage.
